Very simple user interfaces

In a very simple user interface, a form will capture all information in one go. Unfortunately even the simplest domain data models will have attributes that are pointer to other types: for instance a reference to a country in an address.

If the data to choose has a small volume of records, as for countries for instance, one could use a simple drop down. But when there are too many records for a drop down or data needs to be searched, the drop down will not satisfy the bill.

Here I propose a picker for that purpose.
